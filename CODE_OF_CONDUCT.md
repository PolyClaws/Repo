# Code of Conduct

## Our Standards

We are committed to maintaining a focused, professional, and respectful environment for everyone contributing to POLYCLAW.

**Expected behavior:**
- Be direct and constructive in technical discussions
- Respect differing viewpoints and approaches
- Accept feedback professionally
- Focus discussions on the protocol and its development

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Spam, off-topic promotion, or solicitation
- Publishing others' private information
- Conduct inappropriate for a professional environment

## Enforcement

Violations may result in removal from the project's communication channels and/or being banned from contributing.

Report issues to: **conduct@polyclaw.io**

## Scope

This Code of Conduct applies to all project spaces — GitHub issues, pull requests, and any other official POLYCLAW communication channels.
